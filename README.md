# Sonha_Comigo-
Sonha Comigo — uma página-convite romântica feita pra celular, no estilo de um player de música. Tem capa e controles tocando de verdade, um varalzinho com 6 fotos do casal deslizando devagar, e termina com um joguinho: "Aceita sair comigo?" — onde o botão Não fica fugindo até a pessoa topar.
